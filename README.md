# Secret Santa
springboot-jpa-rest-mail-angular

# Instructions 
1- Run the fake mailer server:
  * fakeSMTP-2.0.jar
#
2- Build and Run backend project with command lines: 
  * mvn clean install
  * mvn spring-boot:run
#
3 - Run the frontend project with command:  
  * npm install
  * ng serve
#
4 - Access the application at address: 
  * "http://localhost:4200"
#
